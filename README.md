# customer_shopping_behaviour
Customer Shopping Behaviour Analysis

Project Overview

This project analyzes customer shopping behavior using transactional retail data to identify patterns in purchasing behavior, customer segments, product performance, and subscription engagement.
The objective is to help the business understand what drives customer purchases and how different factors such as discounts, demographics, shipping methods, and loyalty affect sales performance.
The dataset contains approximately 3,900 customer purchase transactions across multiple product categories.

Business Problem

A retail company wants to better understand its customers’ shopping behavior in order to improve sales performance, customer satisfaction, and long-term loyalty.
The management team is interested in identifying trends across:
Customer demographics
Product categories
Discounts and promotions
Customer loyalty and repeat purchases
Subscription adoption

The goal is to answer the following key question:
“How can the company leverage consumer shopping data to identify trends, improve customer engagement, and optimize marketing and product strategies?”

Project Workflow
1. Data Cleaning & Exploratory Data Analysis (Python)

Basic data cleaning and exploratory analysis were performed in Python using Google Colab.
Key steps included:
Handling missing values
Checking data types
Preparing the dataset for SQL analysis

2. Data Analysis (SQL)
Business questions were answered using SQL queries executed in BigQuery.
The analysis focused on understanding:
Customer spending patterns
Product performance
Discount effectiveness
Customer segmentation
Subscription behaviour
